# QuickPrompt (Windows Overlay AI Assistant)

Лёгкое WPF-приложение под Windows в стиле Copilot overlay:
- вызывается из трея или глобальным хоткеем,
- всегда поверх окон,
- отправляет prompt в OpenAI-like API,
- умеет прикреплять скриншот текущего экрана,
- хранит только текущую сессию (без истории чатов на диск).

## Локальный запуск

### 1) Требования

- **Windows 10/11**.
- **.NET SDK 8.0+**.
- Доступ к совместимому API (`/models`, `/chat/completions`) и API key.

Проверка SDK:

```bash
dotnet --info
```

### 2) Клонирование и переход в каталог проекта

```bash
git clone <URL_ВАШЕГО_РЕПО>
cd QuickPrompt
```

### 3) Восстановление зависимостей и сборка

```bash
dotnet restore
dotnet build -c Debug
```

### 4) Запуск

```bash
dotnet run -c Debug
```

После старта приложение работает через системный трей.

## Первичная настройка приложения

1. В трее нажмите **Open/Hide** чтобы показать окно.
2. Нажмите **Settings**.
3. Заполните:
   - **Base URL** (например, `https://api.openai.com/v1`),
   - **API Key**,
   - **Default model**,
   - **Temperature**,
   - **Max tokens**,
   - **Additional headers** (JSON-объект, опционально).
4. Нажмите **Save**.
5. Вернитесь в главное окно и нажмите **↻** для загрузки списка моделей.

> Настройки сохраняются в `%LOCALAPPDATA%/QuickPrompt/settings.json`.
> API-ключ хранится в зашифрованном виде (DPAPI CurrentUser).

## Использование

- Показать/скрыть окно: двойной клик по иконке трея или пункт **Open/Hide**.
- Глобальный хоткей по умолчанию: **Ctrl + Shift + Space**.
- Введите запрос и нажмите **Send**.
- **Сделать скрин** — прикрепляет скриншот экрана к следующему запросу.
- **Вставить буфер** — добавляет текст из clipboard в prompt.
- **Очистить** — очищает текущую мини-сессию в UI.

## Публикация Release (опционально)

```bash
dotnet publish -c Release -r win-x64 --self-contained false
```

Артефакты будут в:

`bin/Release/net8.0-windows/win-x64/publish/`

## Частые проблемы

### `401 Unauthorized`
Проверьте корректность API key и дополнительные headers.

### `404` / ошибки маршрутов
Проверьте `Base URL`: приложение ожидает совместимые пути `/models` и `/chat/completions`.

### Модели не загружаются
Убедитесь, что провайдер поддерживает endpoint `/models` или проксируйте совместимый ответ.

### Глобальный хоткей не срабатывает
Другая программа может уже занять комбинацию `Ctrl+Shift+Space`.
